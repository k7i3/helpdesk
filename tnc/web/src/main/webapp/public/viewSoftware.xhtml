Software
Мониторинг позволяет:
  • наблюдать за местонахождением объектов и их передвижениями на карте;
  • отслеживать изменения скорости движения, уровня топлива;
  • составлять отчеты нескольких видов;
  • отслеживать движение объекта по заданному маршруту;
  • интерпретировать полученную от объекта информацию в виде графика;
  • и многое другое.
За объектами мониторинга можно наблюдать на экране компьютера в режиме «Онлайн».
Пользовательский интерфейс VMS-client прост, существуют всплывающие подсказки, дающие поясняющую информацию к кнопкам и иконкам.
Для начала работы в системе необходимо авторизоваться. Укажите логин и пароль, которые Вам были выданы. После того как Вы ввели все необходимые данные, нажмите кнопку «Ок».

Подраздел «Отображение всего пути» предназначен для просмотра всего пути выбранного объекта за выбранный период времени, с выводом пробега на экрана (см. Рисунок 4)
Для вывода всего пути выбранного объекта необходимо:
  1. Задать интересующий период времени. Для простоты выбора даты в программу встроен календарь, который можно увидеть при нажатии иконки .
  2. При нажатие открывается список Ваших объектов, в котором выбираем интересующий Вас объект и нажимаем на «Ок».
  3. Далее нажимаем «Загрузить данные». Прогресс выполнения данной команды можно отследить на полосе загрузок.
  4. После завершения загрузки данных нажимаем на кнопку «Показать весь путь» и с правой стороны окна на карте видим отображения трека объекта (см. Рисунок 5)
Пробег выбранного объекта за выбранный период показан в левом нижнем углу (см. Рисунок 5).
Также можно вывести на экран график, параметрами которого являются время, наличие данных и скорость.

Рисунок 9 Общий список В подразделе «Журнал сформированных отчетов» мы можем просмотреть сформированные отчеты либо сформировать новый.
Для создания отчета необходимо нажать на иконку ) и выбрать интересующий отчет.

Виды отчетов:
  1. Детальный отчет – отчет по выбранному объекту, в котором указано общее время работы, за указанный период, т пробег. При наличии датчиков уровня топлива – норма расхода и общий расход топлива.
  2. Ежедневный пробег по ТС всех групп ТС – отчет по всем ТС, прикрепленным к филиалу с указание пробега по всем ТС за один день и общий пробег одного ТС за месяц.
  3. Нарушение скоростного режима (детальный отчет).
  4. Нарушение скоростного режима (сводный отчет).
  5. Нарушение скоростного режима по списку ТС (детальный отчет).
  6. По периодам движения для ТС.
  7. Сводный отчет.
 Для формирования отчетов необходимо зайти в "АРМ диспетчера" -> "Журнал сформированный отчетов"
Для формирования Детального отчета необходимо из списка выбрать объект -> выбрать начало периода (всплывающий календарь ) -> выбрать окончание периода (всплывающий календарь ) -> Продолжить.
Если Вы хотите, чтобы отчет открылся сразу как сформируется поставьте галочку «Показывать после формирования».
